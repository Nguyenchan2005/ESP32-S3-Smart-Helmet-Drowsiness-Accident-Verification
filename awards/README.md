# Awards & Competition Evidence

Thư mục này dành để làm nổi bật minh chứng thành tích cuộc thi của dự án.

## Highlight

**ESP32-S3 Smart Helmet** được trình bày như một dự án nghiên cứu có cả đầu ra học thuật và thành tích cuộc thi. Phần này giúp người đọc nhanh chóng nhận ra dự án không chỉ dừng ở nguyên mẫu kỹ thuật, mà còn có giá trị chứng minh qua hoạt động học thuật/thi đấu.

## Minh Chứng Nên Đặt Tại Đây

| Loại minh chứng | Gợi ý tên file |
| --- | --- |
| Giấy chứng nhận top/giải thưởng | `competition-certificate.pdf` |
| Hình ảnh trao giải hoặc poster | `competition-highlight.jpg` |
| Email/thông báo kết quả | `competition-result.pdf` |
| Link bài đăng hoặc thông báo chính thức | Ghi vào mục bên dưới |

## Thông Tin Thành Tích

- Thành tích: Top cuộc thi.
- Dự án: Mũ bảo hiểm thông minh tích hợp HUD, cảnh báo buồn ngủ và SOS.
- Vai trò minh chứng: củng cố độ tin cậy của nghiên cứu khi người đọc xem repository.

## Liên Kết Minh Chứng

Hiện repository đã chuẩn bị cấu trúc để đặt minh chứng. Khi có file chứng nhận hoặc hình ảnh chính thức, hãy thêm vào thư mục này để README chính hiển thị đầy đủ hơn.
