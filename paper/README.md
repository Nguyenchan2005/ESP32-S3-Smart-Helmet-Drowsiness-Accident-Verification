# Accepted Paper

Thư mục này dùng để làm nổi bật đầu ra học thuật của dự án.

## File Chính

| File | Mô tả |
| --- | --- |
| `ACCEPTED_PAPER_SPRINGER_V2.docx` | Bản paper được đặt riêng để người đọc nhận biết nghiên cứu đã có paper accepted. |

## Ghi Chú Kỹ Thuật

File paper gốc được giữ ở định dạng `.docx` để bảo toàn nội dung ban đầu. Trong quá trình kiểm tra tự động, file có dấu hiệu lỗi cấu trúc ZIP/DOCX nên nếu cần chỉnh sửa sâu, nên mở bằng Microsoft Word để Word tự phục hồi định dạng.

## Cách Trình Bày Trong Repo

- README chính đã liên kết trực tiếp đến bản paper này.
- Các bản phục hồi/phụ trợ được chuyển vào `archive/` để thư mục gốc gọn hơn.
- Nếu có thư chấp nhận, email accepted, DOI hoặc thông tin hội nghị/tạp chí, nên bổ sung thêm vào thư mục này dưới dạng PDF/ảnh để tăng độ tin cậy.
